Electric Sheep — Floating Button Style Options

Option A — Neon Outline
buttons

Full-brightness cyan border on every button at all times. Constant soft glow radiates outward. Looks like lit neon tubing. All buttons same colour — clean and unified.

Option B — Colour-coded
buttons

Each button gets its own colour — cyan, purple, mint, pink, amber. Tinted fill + matching border. Instantly tells them apart at a glance.

Option C — Bigger + Pulse
buttons (animated)

Larger size + staggered idle glow pulse. Each button gently breathes in and out at slightly different times, drawing the eye naturally. Hover stops the pulse and locks to bright cyan.
